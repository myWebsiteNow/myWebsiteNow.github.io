Booklend
For our project, named BookLend, we developed a book lending platform that provides students the functionality to share books throughout the student body. We utilized Node.js for our REST API and PostgreSQL for our database. Putting these together, our back-end is both robust and extendable. For our front-end, we used HTML, Bootstrap, and Vanilla JavaScript in an attempt to keep the finished project lightweight and fast.
Olivia Barner
Ceci Kurdelak
Fox Friend Finder
Fox Friend Finder is a group-finding website meant to make it easier to make new friends despite the pandemic environment. It has a calendar that shows what groups are meeting when, discovery tools for finding just the right group, and a chat room for the members of each group. The frontend is a Vue single-page application, and the backend is composed of an ASP.NET Core application and a Postgres database.
Josiah Armstrong
FoxOverflow
We wanted to develop a website where students from all disciplines could collaborate and work together on any number of projects. We took inspiration from the git style open source development and built a website where any number of people could comment, contribute, or create projects that they wanted to work on. We built our website using the LAMP stack hosted on an Ubuntu server. We built a full PostgreSQL database for the site, developed account sign in and security, and implemented the PHP files needed to make everything interact correctly.
